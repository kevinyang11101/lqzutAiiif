# 高校教师科研管理系统 java1112

## 项目概述

高校教师科研管理系统是一款基于Springboot+Vue的前后端分离系统。该系统采用MVC三层架构，通过整合Spring、SpringBoot、MyBatis、Vue和Maven技术栈，构建了一套高效、易用的科研管理解决方案。系统后端使用MySQL数据库存储数据，通过B/S架构实现Web端的操作与管理。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven
- 三层架构：Model-View-Controller（MVC）

## 功能模块

- 教师信息管理
- 科研项目申报与审批
- 成果展示与分享
- 学术论文管理
- 科研经费管理

## 系统角色

- 教师
- 管理员

## 运行环境

- 操作系统：兼容性良好，支持主流操作系统
- Web服务器：Tomcat 8.5及以上版本
- 数据库：MySQL 5.7及以上版本
- 前端：现代浏览器，如Chrome、Firefox等

## 部署步骤

1. 搭建开发环境
   - 安装Java开发工具包（JDK）
   - 安装MySQL数据库
   - 安装Maven构建工具
   - 安装Node.js环境（前端）
2. 克隆代码仓库
   - 获取前端代码
   - 获取后端代码
3. 数据库配置
   - 创建数据库
   - 导入数据表结构
4. 编译与部署
   - 编译前端代码
   - 部署到Web服务器
   - 启动后端服务
